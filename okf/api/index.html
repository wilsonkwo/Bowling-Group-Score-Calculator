API

  • Auth — /api/auth/* login, register, refresh, current user
  • Bowlers — /api/bowlers/* CRUD
  • Sessions — /api/sessions/* sessions and their games
  • Scores — /api/scores/* frame entry and leaderboard

All endpoints except /api/auth/* require a Authorization: Bearer <jwt> header (see JwtAuthFilter). Write operations on bowlers and sessions require ROLE_ADMIN.
