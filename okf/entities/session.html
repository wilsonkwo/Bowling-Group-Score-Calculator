Table: bowling_session

ColumnTypeDescription
idBIGINT (PK)Identity.
session_dateDATE, not nullDate of the outing.
locationSTRINGOptional venue.
notesSTRINGOptional free text.
statusSTRING enum, not nullOPEN (default) or CLOSED.
time_slotSTRING enum, nullableMORNING (8am-12pm), AFTERNOON (1pm-6pm), or EVENING (8pm-12am). Required on create (enforced in the controller, not a DB constraint) — nullable only so sessions created before this field existed don't break.

Relationships

  • One-to-many with Game.

API

Managed via /api/sessions.
