Entities

  • User — login account
  • Role — authorization role (ADMIN, USER)
  • Bowler — a player, optionally linked to a User
  • BowlingSession — a date/location grouping of games
  • Game — one game within a session
  • BowlerGame — a bowler's participation/result in a game
  • Frame — one of the 10 frames within a BowlerGame
